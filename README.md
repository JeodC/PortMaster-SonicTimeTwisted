## Installation
Ready to run! Uses [gmloader](https://github.com/JohnnyonFlame/droidports) and [SonicTimeTwisted](https://github.com/overbound/SonicTimeTwisted) open source repository.

## Default Gameplay Controls
| Button | Action |
|--|--|
|START|Pause / Accept|
|D-PAD|Move|
|LEFT ANALOG|Move|
|ABXY|Jump / Fly|
|DOWN + ABXY|Spindash|
