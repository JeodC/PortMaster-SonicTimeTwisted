## Installation
Download the Windows version of the game from [Gamejolt](https://gamejolt.com/games/sonic-time-twisted/24775) and place the entire `.zip` in `ports/sonictimetwisted` without extracting.

## Default Gameplay Controls
| Button | Action |
|--|--|
|START|Pause / Accept|
|D-PAD|Move|
|LEFT ANALOG|Move|
|ABXY|Jump / Fly|
|DOWN + ABXY|Spindash|
